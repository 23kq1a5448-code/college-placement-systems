# College Placement System

A static front-end project for a college placement portal. It includes:

- Login and registration flow
- Student profile page
- Dashboard with 30 company listings
- Apply functionality and my application tracking
- Logout flow

## Run locally

1. Open `index.html` in a browser.
2. Register a student account.
3. Login and explore the dashboard.

## Notes

- Student credentials and application state are stored in browser `localStorage`.
- This is a static HTML/CSS/JavaScript project and does not require a backend server.
